### Install

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3 
```

(Optional)

```bash
sudo apt install python3-pip
pip3 install virtualenv
python3 -m virtualenv env
source env/bin/activate
```

(Continue)

```bash
pip3 install -r requirements.txt 
```
